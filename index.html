What this is

Beg2DFIR is a public workbench for learning technical skills from the ground up. The goal is simple: study the fundamentals, build useful things from them, and leave a trail that another beginner can actually follow.

This site collects the written side of that process: notes, projects, study packs, puzzles, and anything else that turns learning into something visible.

The name means Beginner to DFIR, but the path does not start with forensics. It starts with the basics: devices, networks, operating systems, troubleshooting, security, and the discipline to keep going when the work is not exciting yet.
